# Custom-Upload-On-AWS-S3-Based-On-Storage-Type
Custom Upload on AWS S3 Based on Storage Type

Storage Type = 1 : Upload on AWS S3 Bucket<br/>
Storage Type = 2 : Upload on Local System<br/>
Storage Type = 3 : Upload on Both Local System & AWS S3 Bucket<br/>
